微助后台管理系统1.2.0

2016年05月5日
1.完成线下培训功能模块，超级管理员可以在发现板块中配置“线下培训”.
2.将“线下培训”板块功能假如角色权限管理中；需要在管理员角色中加上“线下培训”的管理员角色之后才能对线下培训中的信息进行管理操作.
3.修复“调研管理”和“发现管理”中的部分BUG

微助后台管理系统1.1.0

2016年04月8日
1.考试管理-试卷管理-查看结果添加整体概况统计，可以按部门统计、按职务统计、按考题统计；可以查看参考人员的答卷详情。
2. 调研部分提供调研问卷的预览功能。

微助后台管理系统1.0

2016年03月21日
1.新后台增加权限管理，可以创建多个角色 (注意：每个角色可以分配不同的权限，系统默认的角色供参考，可修改和新建)。
2. 新后台可为不同管理员分配不同角色和部门管理。 (注意：新建管理员时，可选择多个角色，多个部门)
3. 新后台模块顺序和界面风格有调整，功能不变，有需帮助可点击右上角“帮助”，查看《微助产品后台使用文档》。
